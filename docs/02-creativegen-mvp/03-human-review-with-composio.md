# Flow 3 – Human Review via Composio (Deprecated for MVP)

**Status:** Deprecated for MVP. Kept as an optional future extension.

**MVP approach:** Human creative QA, review, and approval are implemented **in-app**. See [03-human-review-in-app.md](03-human-review-in-app.md) for the current implementation plan.

**Optional later:** This document described routing review through Composio → Slack (for reviewer feedback) and Notion (as system of record). That path can be revisited as a post-MVP extension if teams need review inside Slack/Notion; the app would remain the source of truth, with Composio used only to mirror or trigger from external tools.
